# What is SQL?

**SQL** stands for **Structured Query Language**. It is a language that is used to talk with databases.

When we want to interact with data stored in a database — whether to fetch it, add new data, update existing data, or delete it — we use SQL to do so. Just like we use English to communicate with people, we use SQL to communicate with databases.

## What is a Database?

A database is a place where data is stored in an organized way.

There are different database softwares available, such as:

- MySQL
- PostgreSQL
- SQL Server
- Oracle

Every company stores its data in databases because:

- Databases are **secure**
- They can **handle massive amounts of data** efficiently
- They allow **multiple users** to access data at the same time

## Simple Example

If we want to fetch the names of all customers from a database, we write a SQL query like:

```sql
SELECT Name FROM Customers;
```

The database understands this instruction and returns the list of customer names.

## Key Takeaway

> SQL is the standard language used worldwide to interact with relational databases. It is one of the most essential skills for any data analyst.
