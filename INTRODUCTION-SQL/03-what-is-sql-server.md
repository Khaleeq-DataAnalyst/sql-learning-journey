# What is SQL Server?

**SQL Server** is the name of a DBMS software created by **Microsoft**. It runs on a server and uses the SQL language to provide database services to clients.

## Breaking Down the Name

- **SQL** → the language used to write queries
- **Server** → it provides services to other computers (clients)
- (Microsoft) → the company that created it

## What is a Server?

A **server** is a powerful computer that provides services to other computers.

For example, when you open YouTube and watch a video — your laptop is the **client**, and YouTube's server (a powerful computer) sends the video to you.

### Server vs Normal Computer

| Normal Computer | Server |
|---|---|
| Used by one person | Used by thousands at the same time |
| 8–16 GB RAM | 128–512 GB RAM |
| Can be shut down anytime | Runs 24/7 without stopping |
| Kept at home or office | Kept in a data center |

## How SQL Server Works

```
YOUR LAPTOP                    OFFICE SERVER
(Client)                       (Server Machine)
   │                                  │
   │   SQL Query                      │
   │ ─────────────────────────────▶   │
   │                                  │  SQL Server
   │                                  │  processes query
   │                                  │  and gets data
   │                                  │  from database
   │ ◀─────────────────────────────   │
   │   Result                         │
```

## Important Note

There are **two meanings** of the word "server":

- **Hardware server** → the physical computer machine
- **Software server** → a program that provides services (like SQL Server)

SQL Server is actually a **software**, but it runs on a **hardware server**.

## Why Microsoft SQL Server is Popular

SQL Server is widely used in:

- Corporate companies
- Banks and finance institutions
- Healthcare systems
- Microsoft ecosystem-based companies

## Key Takeaway

> SQL Server is Microsoft's DBMS software that runs on a server machine and uses SQL language to manage and serve database services to clients over a network.
